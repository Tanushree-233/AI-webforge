# AI-webforge

Tanushree Rathod Portfolio

Overview

This is a personal portfolio website created to showcase my skills, projects, and experience as a Data Science student. The website highlights my work in Python, SQL, Machine Learning, and data-driven applications.

The portfolio includes sections for introduction, skills, projects, GitHub activity, and contact information. It is designed with a a modern dark user interface and smooth animations.

Features

- Responsive portfolio website
- Hero section with introduction
- About section describing background and interests
- Skills section highlighting technical abilities
- Projects section showcasing selected work
- GitHub contribution graph
- Contact section with email and social links
- Scroll animations and modern UI design

Technologies Used

- HTML
- CSS
- JavaScript
- AOS Animation Library
- GitHub Contribution Graph
- Particle.js for background animation

Projects Included

Loan Approval Prediction Chatbot
A chatbot system built using LangChain and HuggingFace that allows users to query loan approval data using a CSV dataset.

Traffic Risk Analysis
A machine learning project that analyzes traffic data to identify potential accident risks and predict high-risk areas. The model helps understand traffic patterns and supports better road safety planning.

Twitter Sentiment Analysis
A natural language processing project that analyzes tweets to determine sentiment such as positive, negative, or neutral. This project uses machine learning techniques to understand public opinion from social media data.

Deployment

The portfolio website can be deployed using platforms such as Vercel, Netlify, or GitHub Pages.

Deployment steps:

1. Upload the project to GitHub.
2. Connect the repository to Vercel.
3. Deploy the project to generate a public website link.
